# Deep Research Team

Use this preset for thorough investigation across multiple dimensions.

## Team Composition
- **Team Lead**: orchestrator (decomposes question, synthesizes findings)
- **Teammates**: 2-3 researcher instances

## Workflow
1. **Decompose** — Orchestrator breaks the research question into independent sub-questions
2. **Investigate** — Researcher teammates each tackle a sub-question in parallel
3. **Synthesize** — Orchestrator combines all findings into a coherent analysis

## Task Dependencies
```
all researcher tasks (parallel) → orchestrator synthesis
```

## Plan Approval
Not required — research is non-destructive.

## Example Invocation
```
Create an agent team to research [topic].
Use the research preset: split the question into independent angles
and have multiple researchers investigate in parallel.
```
