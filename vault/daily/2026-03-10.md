## Summary: Added Wave Function Collapse (WFC) as 6th Genome Type

**What**: Implemented a full Wave Function Collapse generative algorithm that produces constraint-satisfying tile patterns — fundamentally different from the existing 5 genome types.

**How WFC works**:
1. Each cell starts as a superposition of all possible tile types
2. The cell with lowest Shannon entropy is selected and collapsed to a single tile
3. Constraints propagate to neighbors (only tiles allowed by adjacency rules survive)
4. Repeat until all cells are resolved
5. Optional symmetry (horizontal/vertical/quad) is applied post-collapse

**Key features**:
- `WFCRule`: `tileCount` (3-8), `adjacency` matrix, `weights`, `symmetry`
- 3 seed genomes with different topologies: gradient chain, fully connected, ring
- Mutation: flip adjacency connections, perturb weights, change symmetry, add/remove tiles
- Crossover: merges adjacency graphs from both parents with random pruning
